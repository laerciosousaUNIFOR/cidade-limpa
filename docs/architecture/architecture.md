# Arquitetura do Sistema

## Descrição Geral
O sistema segue arquitetura cliente-servidor baseada em APIs REST.

## Componentes

- Frontend Web (React)
- Frontend Mobile (React Native)
- Backend (Node.js + Express)
- Banco de Dados (PostgreSQL)

## Padrão Arquitetural
MVC (Model-View-Controller)

## Comunicação
O frontend se comunica com o backend via HTTP/JSON.

## Justificativa das Tecnologias
- React: alta performance e reutilização
- Node.js: escalabilidade
- PostgreSQL: robustez e confiabilidade

## Protótipos
Os protótipos foram desenvolvidos no Figma para validação de UX/UI.
