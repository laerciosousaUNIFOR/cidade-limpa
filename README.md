# Cidade Limpa

## Descrição
O Cidade Limpa é um sistema multiplataforma (web e mobile) desenvolvido para permitir que moradores registrem, acompanhem e colaborem na resolução de problemas urbanos, como acúmulo de lixo, buracos e falhas na iluminação pública.

A solução busca melhorar a comunicação entre a comunidade e responsáveis locais, promovendo uma gestão mais eficiente e sustentável.

## Problema
A falta de canais eficientes para comunicação de problemas urbanos resulta em acúmulo de lixo, degradação ambiental e baixa qualidade de vida nas comunidades.

## Objetivo
Desenvolver uma plataforma digital que permita o registro, acompanhamento e gerenciamento de problemas urbanos de forma simples, rápida e acessível.

## Escopo
- Cadastro e autenticação de usuários
- Registro de problemas com imagem e localização
- Acompanhamento de status
- Painel administrativo
- Notificações de atualização

## Tecnologias
- Frontend Web: React
- Frontend Mobile: React Native
- Backend: Node.js (Express)
- Banco de Dados: PostgreSQL

## Arquitetura
O sistema segue o padrão cliente-servidor com APIs REST, utilizando arquitetura MVC.
